  + 111年12月20日地球科學院111學年度第4次主管院務座談會議通過

  第一條、為維持中央大學地球科學學系大地電磁（Magnetotelluric，簡稱MT）長期觀測站之運作，並達到資料保存、服務及共享之目的，特訂本管理辦法（以下稱本辦法）及收費標準。

### 2023-06-21赴野外現場，整理目前狀況
+ 資料回傳相關問題處理:
  + 4G設備現在硬體沒有問題，訊號狀況如果持續不佳要移動到訊號更好的位置。
  + 回傳機制:
    + MTU主機->(使用rsync服務透過固定IP的4G路由器)->海大。回傳成功後檔案將會從MTU主機刪除，所以SD卡檔案幾乎不會滿。但log檔會不斷增大，不會刪除。
    + SD卡設定檔包含回傳伺服器的IP、Port、ssh金鑰。安全性是靠海大的防火牆，所以整個系統只有海大指定IP可以遠端連入做很有限的操作。
    + SD卡要配合海大伺服器、4G路由器各項設定來寫設定檔，已經寫好的SD卡原則上不用動。 -> 每次最好不要更換SD卡，若要更換SD卡要特別留意設定檔是否正確。
    + 其他使用者去海大伺服器用FTP軟體下載檔案。好的FTP軟體可以忽略下載過的檔案。
    + 若訊號只有3G，表示網路狀態很差，上傳速度極低，造成大量資料堆積在MTU主機。但回到4G後應該可以快速消化資料。
    + rsync回傳機制不能控制先回傳哪個檔案，因此大量堆積時看起來檔案可能會缺漏，但整個傳完之後會補滿。
+ 電源相關問題處理:
  + 線路非常亂，除了MTU主機還供應電給姜智文的主機。
  + 使用明緯企業DRC-100A製作了UPS效果的環境，平時是AC轉DC直接供電13.8[V]，並對車用電池充電，當AC不穩定或中斷時將切去備用車用電池，兩顆電池預計可維持MTU主機3天，但有另一台所以會更少一些。
  + 防雷擊線路: 據姜智文轉述有3級，要怎麼接線不清楚。
